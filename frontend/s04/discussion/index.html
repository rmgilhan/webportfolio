HTML Images

An image can be shown using < img > tag with its src attibute

<img src="https://i.imgur.com/QmvceUy.png">

<img src="./images/zuitt.png"/ >

Valid image extensions are the following:

  1. png
  2. jpg
  3. gif
  4. svg
  5. webp

Take a look at w3schools attribute references
